# TravelWith

A Spring Boot application that connects students traveling on the same routes, helping them find travel companions for safer and more economical journeys.

## About
TravelWith is a Spring Boot application that connects students traveling on the same routes. Users enter their travel details—name, phone number, source station, destination, date, and time—and the app displays a list of fellow travelers on the same train or station. This helps students find travel companions, share expenses, and travel safely together.

## Features
- User registration with travel details
- Travel companion matching based on:
  - Source station, date of travel and train departure time
  - Train no. and date of travel

- Display list of fellow travelers on the same route
- Phone number sharing for coordination
- Simple and user-friendly interface

## Tech Stack
- **Backend:** Spring Boot
- **Database:** MySQL
- **Frontend:** HTML, CSS
- **Template Engine:** Thymeleaf

## Prerequisites
- Java 8 or higher
- MySQL Server
- Maven
